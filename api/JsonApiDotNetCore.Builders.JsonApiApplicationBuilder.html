Class JsonApiApplicationBuilder

A utility class that builds a JsonApi application. It registers all required services and allows the user to override parts of the startup configuration.

ConfigureMvc()

Configures built-in .NET Core MVC (things like middleware, routing). Most of this configuration can be adjusted for the developers' need. Before calling .AddJsonApi(), a developer can register their own implementation of the following services to customize startup: IResourceGraphBuilder, IServiceDiscoveryFacade, IJsonApiExceptionFilterProvider, IJsonApiTypeMatchFilterProvider and IJsonApiRoutingConvention.
