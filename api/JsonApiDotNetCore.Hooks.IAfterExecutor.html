AfterCreate<TEntity>(IEnumerable<TEntity>, ResourcePipeline)

Executes the After Cycle by firing the appropiate hooks if they are implemented.

Fires the AfterCreate(HashSet<T>, ResourcePipeline) hook where T = TEntity for values in parameter entities.

Fires the AfterUpdateRelationship(IAffectedRelationships<T>, ResourcePipeline) hook for any related (nested) entity for values within parameter entities
